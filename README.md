# TechFuture 2026

Landing page desenvolvida com HTML e CSS para divulgação de um evento de tecnologia fictício.

---

## Sobre o projeto

O **TechFuture 2026** é um site institucional que simula a divulgação de um evento de tecnologia, contendo informações sobre programação, ingressos e contato.

O objetivo do projeto foi aplicar conceitos de:
- Estruturação com HTML
- Estilização com CSS
- Organização de arquivos
- Navegação entre páginas

---

## Estrutura do site

O projeto possui 5 páginas:

- **Início** → apresentação do evento  
- **Sobre** → informações gerais  
- **Programação** → agenda do evento  
- **Ingressos** → tabela de preços  
- **Contato** → formulário de contato  

---

## Tecnologias utilizadas

- HTML5  
- CSS3
